# SaveBite - Food Waste Management System

## 📌 Project Description
SaveBite is a Food Waste Management System developed to reduce food wastage by connecting restaurants with customers.

## 🚀 Technologies Used
- HTML
- CSS
- Bootstrap
- JavaScript
- Python (Flask)
- MySQL

## ✨ Features
- User Registration & Login
- Restaurant Registration
- Add Food Items
- Browse Available Food
- Book Food
- Owner Dashboard
- Restaurant Dashboard

## ▶️ Run the Project

```bash
pip install -r requirements.txt
py app.py
```
